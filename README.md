CRAPtion
========

Requirements
------------
Python packages
```
configobj
dropbox
```
Config
```
cd
git clone https://github.com/spillevink/CRAPtion.git
echo export PATH=$path:~/CRAPtion >> .bash_profile
./CRAPtion/craption
vim .craptionrc
```
Dropbox login
```
python CRAPtion/settings.py dropbox
```
Restore default settings
```
python CRAPtion/settings.py clear-conf
```
